# 🧠 K-Means & DBSCAN Clustering Project

> A project demonstrating **unsupervised machine learning techniques** to discover patterns and group data effectively 📊

---

## 🚀 Project Overview

This project implements the **K-Means Clustering Algorithm from scratch** without using built-in machine learning libraries.

It also explores **DBSCAN**, which is more suitable for:

* Handling noisy datasets
* Detecting clusters of irregular shapes

The project uses real datasets like student data and restaurant data to demonstrate clustering performance.

---

## ✨ Key Features

* 🔢 K-Means algorithm from scratch
* 📊 Data visualization using Matplotlib
* 📁 Multiple dataset support
* 🔁 Iterative centroid optimization
* 📉 Convergence using tolerance
* 🔍 DBSCAN implementation

---

## 🧠 Algorithm Insight

**K-Means Steps:**

1. 🎯 Initialize centroids randomly
2. 📏 Calculate distance from points
3. 📌 Assign clusters
4. 🔄 Update centroids
5. ⏹️ Repeat until convergence

## 🔬 Research Insight

* K-Means uses **Euclidean distance**, which is not ideal for geographical data
* Better alternatives:

  * 🌍 Haversine Formula
  * 📐 Vincenty Formula
